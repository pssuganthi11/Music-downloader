How to use this script?

Clone this repository and open Terminal & go to this directory using: 

git clone <repository-url>
cd tamil-mp3-downloader
Requirements

Ensure you have Python 3+ installed. Check by running: 

python -v  # prints python version
pip -v     # checks if pip is installed

If Python is not installed, install it (refer to Step 2 of installation guide).

Install Dependencies
pip install -r requirements.txt
Run the Python Script
python main-classic.py
